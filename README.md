# Logistics SQL Analysis

## Overview
This project focuses on analyzing logistics operations and shipment data using SQL.

The goal is to transform raw data into clear, actionable insights to support better decision-making in logistics and supply chain processes.

---

## Dataset
The dataset includes information about shipments, routes, transport types, and delivery statuses.

Main tables:
- shipments
- routes

---

## Key Analysis

The analysis covers:

- Total number of shipments
- Total and average shipment weight
- Shipments distribution by delivery status
- Distribution by transport type
- Top origin countries
- Top destination countries
- Delivery zones analysis
- Longest routes identification
- Average distance by transport type

---

## Key Insights

- Road transport dominates logistics operations
- Germany and Italy act as major logistics hubs
- Western Europe has the highest shipment volume
- Long-distance routes are mostly international
